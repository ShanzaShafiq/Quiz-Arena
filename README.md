# 🧠 Quiz Arena

An interactive, responsive quiz web app with multiple categories, instant scoring, answer review, and a dynamic leaderboard system.

---

## 🚀 Features

* 🎯 Multiple quiz categories (HTML, CSS, JS, C, C++)
* ⚡ Real-time MCQ gameplay with timer
* 🟢 Instant correct/incorrect feedback
* 📊 Score tracking system
* 🏆 Leaderboard with top scores (stored locally)
* 🔍 Detailed review of all answers after quiz
* 📱 Fully responsive UI (mobile + desktop)
* 🌙 Light/Dark mode toggle
* 🎉 Smooth animations & modern UI effects

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Advanced styling + animations)
* JavaScript
* LocalStorage (for leaderboard & progress)
* Chart.js (performance graph)
* Confetti.js (celebration effects)

## 🎮 How to Run

1. Clone or download the project
2. Open `index.html` in your browser
3. Select a category and start the quiz
